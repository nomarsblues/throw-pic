# throw-pic
## Introduction
a image uploader based on [sm.ms](https://sm.ms/)
## Start
```
go build
```
```
./throw-pic -p ${local_path}
```
## Usage
### Integrated with [Typora](https://typora.io/)
Preferences -> Image -> Image Uploader
